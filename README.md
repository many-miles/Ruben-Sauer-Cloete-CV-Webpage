# Ruben Sauer Cloete - CV Webpage

A clean, professional online CV/resume built with HTML and CSS.

## Overview

This project is a web-based version of my professional CV, created to provide an accessible, shareable online presence. The webpage presents my education, experience, skills, and projects in a format optimized for both viewing and printing.

## Live Version

[View CV Webpage](https://many-miles.github.io/Ruben-Sauer-Cloete-CV-Webpage/)

## Purpose

The primary goals of this project are:
- Provide an always-accessible version of my CV
- Create a shareable link for applications and networking
- Practice HTML/CSS fundamentals
- Demonstrate basic web development skills
- Offer a printable version for offline use

## Features

- **Clean, Professional Design**: Simple, readable layout focused on content
- **Responsive Layout**: Adapts to different screen sizes
- **Print-Optimized**: CSS print styles for professional printed output
- **Semantic HTML**: Proper document structure and accessibility
- **Fast Loading**: Minimal assets for quick page load
- **Easy Updates**: Simple structure for content updates

## Technical Implementation

### Tech Stack
- HTML5
- CSS3
- GitHub Pages hosting

### Design Approach

**Layout:**
- Single-page design for easy scanning
- Clear section hierarchy
- Professional typography
- Adequate whitespace for readability

**Styling:**
- CSS Grid or Flexbox for layout
- Print-specific media queries
- Mobile-responsive breakpoints
- Professional color scheme

**Structure:**
```html
<header>
  <h1>Name</h1>
  <nav>Contact Info</nav>
</header>
<main>
  <section id="summary">Professional Summary</section>
  <section id="experience">Work Experience</section>
  <section id="education">Education</section>
  <section id="skills">Technical Skills</section>
  <section id="projects">Projects</section>
</main>
<footer>Additional Info</footer>
```

### Viewing
Simply visit the GitHub Pages URL to view the CV online.

### Printing
Use your browser's print function (Ctrl+P / Cmd+P) to generate a PDF or physical copy.

### Sharing
Share the GitHub Pages URL in applications, emails, or on LinkedIn.
